# Task Planner

A simple, elegant task management application with plans for Google Calendar integration.

## Features

- Add tasks with name, date, and time
- Filter tasks by: All, Today, Upcoming, Completed
- Mark tasks as complete/incomplete
- Delete tasks
- Responsive design
- Local storage persistence

## Coming Soon

- Google Calendar integration
- Task categories
- Recurring tasks
- Task priorities

Visit the planner at: https://krishkc5.github.io/planner/
